# 🎓 Student Management System

A full-stack web application to manage student records with authentication.

---

## 🚀 Features

* 🔐 User Signup & Login (Authentication)
* 📊 Add, View, Update, Delete Students (CRUD)
* ✅ Form validation
* 🎨 Clean UI

---

## 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MySQL
* Axios

---

## 📂 Project Structure

* frontend/
* backend/
* database/

---

## ▶️ Run Project

### Backend

cd backend
npm install
node server.js

### Frontend

cd frontend
npm install
npm start
